# Chronologie des mathématiques

Une frise chronologique interactive pour visualiser l'histoire des mathématiques et consulter des fiches courtes sur les mathématicien·ne·s et découvertes.

## Exemple de donnée
Extrait simplifié du format utilisé (fichier data.json) :

```json
{
  "periodes": [
    {
      "Year": -625,
      "End Year": -545,
      "Headline": "Thalès de Milet",
      "Text": "<p><b>Thalès de Milet</b> (vers 625 - vers 545 av. J.-C.) est souvent considéré comme le premier philosophe et mathématicien de la Grèce antique.</p>",
      "Articles URLs": null,
      "Articles Sujets": null,
      "Media": "https://upload.wikimedia.org/wikipedia/commons/thumb/c/c6/...jpg",
      "Media Caption": "Source : Thalès (Wikimedia Commons)"
    }
  ]
}
```

Veillez à respecter la structure (clé "periodes" contenant des objets avec Year, End Year, Headline, Text, Media, etc.) lors de l'ajout ou de la modification d'entrées.
